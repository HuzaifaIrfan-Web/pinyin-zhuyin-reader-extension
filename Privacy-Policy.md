# Privacy Policy

This extension (“Pinyin Zhuyin Reader”) does not collect, store, or transmit any personal data.

It only supports phonetic input (Pinyin and Zhuyin), not Chinese characters.
Click on Pinyin or Zhuyin text to instantly hear pronunciation. It helps learners practice accurate pronunciation while browsing. 

No data is sent to external servers.

No personally identifiable information, browsing history, or user activity is collected or shared.

All functionality is performed entirely on-device.

---

## Data Usage
- All processing happens locally in the browser
- No user data is uploaded or transmitted anywhere
- No analytics, tracking, or telemetry is used

---

## Audio Files
All pronunciation audio files are included locally within the extension package.

These audio files were originally obtained from educational sources such as:
- https://www.yoyochinese.com  
- https://www.dong-chinese.com  
- https://www.archchinese.com

The files are bundled with the extension and are played locally without making any external network requests.

---

## Third-Party Services
This extension does not use any third-party analytics, tracking services, or external APIs.

---

## Security
Since all functionality runs locally inside the browser, no user data leaves the device.
